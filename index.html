基于hexo搭建Next主题博客

发表于 2023-01-23 更新于 2023-01-24 分类于 tech

新的一年，开始自己搭建博客，写一些我认为比较重要的内容。

现在，把我搭建博客遇到的问题写下来并提供解决方案。

安装Node.js git hexo next主题就不再进行描述了，这些步骤简单。

照着网上的操作步骤来做，能够解决百分之七八十的内容。但是还有的内容就需要自己琢磨。下面把我遇到的问题和解决方法描述进行描述。

  • 在站点配置文件中，即使设置了语言属性，但是展示还是乱码。

    可能原因是编码不同，原配置文件是‘GB2312’,改成’utf-8‘就可以了。

  • 安装完主题并运用，没有标签和分栏页。


在主题目录下找到设置运行的主题名的文件夹，进入，打开‘_config.yml’文件，找到“menu”，在此设置要用的不同目录名。

  • npm install hexo-deployer-git –save 失败
    ERROR Deployer not found: git

 
  使用cnpm代替npm安装hexo-deployer-git –save

  • hexo博客布置到github，图片不显示

npm install https://github.com/CodeFalling/hexo-asset-image --save

站点配置文件post_asset_folder属性置为true

npm install hexo-image-link --save

如果npm不行，就换cnpm
windows上复制的路径是 “//..” 改成 linux上的路径 “./..”


成功显示！

  • page无法正常显示

在博客根目录下：

hexo new page "Name"
之后打开”source/Name/Name.md”文件，在文件内添加
type: "Name" 即可。
